const KArchiveFile* KArchiveDirectory::file ( const QString &  name) const

Returns the file entry in the archive with the given name. If the entry exists and is a file, a KArchiveFile is returned. Otherwise, a null pointer is returned. This is a convenience method for entry(), when we know the entry is expected to be a file.
