Detailed Description

The system information class.

This class provide access to operating system's information via CInformer interface; you could get extra system information via CConnector interface. Besides, you can CSysInfo::randBuffer retrieve a randomized buffer in a byte array. The main page of DSSysInfo library help shows how to create an instance of this class.
